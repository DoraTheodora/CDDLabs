  6. List of files:
    • main.cpp - the main process from where the programs starts
    • semaphore.h - the SEMAPHORE header class, for managing threads
    • semaphore.cpp - the SEMAPHORE class, for managing threads with implemented function
    • SafeBuffer.h- the SafeBuffer header class, that defines the abstract methods that need to be implemented
    • SafeBuffer.cpp - the SafeBuffer class, that implements the logic of the queue data structure
    • makefile - the script for compiling the program
  7. Detailes:
    1. This lab goal is to syncronize N threads in such a way, so that producers are able to write data to a data structure before consumers are allowed to read from the data strucure
    2. How the goal was achieved
      • SafeBuffer class implements a classic queue
      • Semaphore class defines the semaphores objects
      • main contains the threads, that need to be ordered in their execution
