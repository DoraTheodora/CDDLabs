Detailed Description

In this file the threads are created and manipulated in such a way, that consumers can read and manipulate data from producers, only and only if data is present.

Using C++ Sempahores, and a collectivectors of threads, a solution was implemented for the Producer-Consumer Problem. The safe buffer was designed as queue with the FIFO rule: First element in is the first element out.
